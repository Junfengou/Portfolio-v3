## To run it locally on your PC

1. Download the file

2. Install required dependencies

   ```sh
   npm install
   ```

3. Start the development server

   ```sh
   npm start
   ```

4. Master is now locked

   ```sh
   To change this, go to -> Settings -> Branches -> Branch Protection Rules
   ```
